# Peg Solitaire Offline Reinforcement Learning Pipeline

This directory (`ss_game_rl`) contains the Python-based Machine Learning environment designed to train an AI on Peg Solitaire using data collected from the Next.js web application.

## Overview

The web application (hosted on Vercel) records human gameplay trajectories (state-action pairs) and pushes them to an Upstash Redis list (`rl:trajectories`).
This local environment pulls that data, supplements it with synthetic data, and trains a PyTorch Neural Network using Behavioral Cloning / Offline RL.

## Directory Structure

*   `requirements.txt`: Python dependencies needed for the project.
*   `fetch_data.py`: Connects to Upstash Redis to download raw JSON trajectories generated by human players on the web app.
*   `environment.py`: A lightweight NumPy-based Peg Solitaire simulator. Used to generate synthetic trajectories through random play (or future MCTS) to supplement human data.
*   `train.py`: Defines the PyTorch Convolutional Neural Network (CNN) Policy model and the training loop for Behavioral Cloning.
*   *Planned:* `evaluate.py`: To test the trained model against the environment.
*   *Planned:* `export_onnx.py`: To export the trained `.pth` model to `.onnx` for deployment in the Next.js web app.

## Quick Start

### 1. Setup Environment

It is recommended to use a virtual environment:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 2. Configure Environment Variables

The `fetch_data.py` script requires access to your Upstash Redis instance.
Create a `.env` file in this directory with your credentials (found in your Vercel/Upstash dashboard):

```env
UPSTASH_REDIS_REST_URL="https://<your-upstash-url>"
UPSTASH_REDIS_REST_TOKEN="<your-upstash-token>"
```

### 3. Fetch Human Data

Run the fetch script to pull all recorded games from Vercel into a local `trajectories.json` file.

```bash
python fetch_data.py
```

### 4. Train the Model

Run the training script to load the JSON trajectories into PyTorch Tensors and train the Policy Network.

```bash
python train.py
```

This will produce a `peg_solitaire_policy.pth` weight file containing your trained model.
